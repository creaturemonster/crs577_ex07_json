Inventory Management Web Service

This is an updated version of the Inventory Management page seen earlier. It has been written using jQuery to simplify the JavaScript and JSON for the interactions with the Web service

This page issues HTTP GET, PUT, POST, DELETE requests based on user interactions to a simple inventory store. The pages uses javascript's XMLHttpRequest to issue these HTTP requests. The inventory store is implemented as a RESTful web service and the inventory items are accessed and updated as URL resources.

This page requires JSON be enabled on the server. You must complete part 2 of the exercise or deploy the solution from the project: Ex81_jaxrs_solution_JSON
